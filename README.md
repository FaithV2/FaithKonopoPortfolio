# Faith Konopo  Portfolio

A React + Vite + Tailwind single-page portfolio website (dark theme) for Faith Konopo Mojanaga.

## Features
- Single Page Application (SPA) with smooth scrolling sections
- Dark black/gray aesthetic
- Sections: Home, About, Skills, Experience, Projects, GlamLink, Contact
- Contact form uses `mailto:` to open the user's email client
- Includes Masteka Production Dashboard project entry
- Ready to run with `npm install` and `npm run dev`

## Quick start
1. `npm install`
2. `npm run dev`
3. Open the local dev URL shown by Vite.

## Notes
- Replace placeholder images in `src/assets/` with your real screenshots.
- Update project links in `src/data/projects.js` if you host demos or repos.
